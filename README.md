**Python-Keylogger**
A Python-based keylogger that captures real-time keystrokes and mouse clicks/movements using the Pynput library. Built for **educational purposes** to understand system-level input monitoring and cybersecurity concepts.

**📌 Features**
  ->Captures real-time keystrokes and logs them to a file
  ->Tracks mouse clicks and movements
  ->Handles special keys — Space, Shift, Caps Lock, Enter
  ->Modular code structure for keyboard and mouse separately
  ->Logs stored in log.txt
  
**📁 Project Structure**
  📄 main.py  —  Keyboard listener & keystroke logger
  📄 listen.py  —  Mouse movement tracker
  📄 control.py  —  Keyboard & mouse controller (simulation)
  📄 append.py  —  File append utility
  📄 log.txt  —  Output log file

**⚠️ Disclaimer**
This project is **built strictly for educational purposes**. Do not use it on any system without explicit permission. Unauthorized use of keyloggers is illegal and unethical.
